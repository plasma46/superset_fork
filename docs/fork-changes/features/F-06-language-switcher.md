# F-06 — Переключатель языка RU/EN

**Дата:** 2026-05  
**Статус:** ✅ Задеплоено  
**Коммит:** `7181568913`

## Что добавлено

Кнопка переключения языка интерфейса (RU ↔ EN) в навигационном баре Superset. Смена языка происходит без перезагрузки страницы через `i18n` Superset.

## Файлы

| Файл | Изменение |
|------|-----------|
| `superset-frontend/src/components/Menu/MenuRight.tsx` | Добавлена кнопка переключения языка |
| `superset/views/core.py` | Эндпоинт для смены языка (если добавлялся серверный) |

## Реализация

Переключатель использует механизм `LANGUAGES` из конфигурации Superset:

```python
# superset_config.py
LANGUAGES = {
    "en": {"flag": "us", "name": "English"},
    "ru": {"flag": "ru", "name": "Русский"},
}
```

Выбранный язык сохраняется в cookie `locale`, который читается при следующей загрузке.

## UX

- Кнопка расположена в правой части навбара
- Отображает текущий язык как флаг или аббревиатуру
- Клик — немедленное переключение
