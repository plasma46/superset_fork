## Что добавлено

Механизм навигации между дашбордами с передачей значений фильтров через URL. Кликабельные ссылки в ячейках таблицы, которые открывают целевой дашборд с предустановленными native filters.

## Подход

Используется Jinja-шаблонизация в SQL датасета. Ссылки строятся через `{{ url_for(...) }}` или статические URL с параметрами фильтров в формате, который Superset понимает для pre-populated native filters.

## Формат ссылки

```
/superset/dashboard/<slug>/?preselect_filters={"<filter_id>":{"<column>":"<value>"}}
```

Для передачи нескольких фильтров значения разделяются через pipe `|` в SQL-выражении датасета, и парсятся на стороне целевого дашборда.

## Файлы
